# Chess-js

## Stack
App created with:
* Node.js
* Socket.io
* Express.js
* Chessboard.js
* Chess.js


## Demo
Deployed on Heroku: https://barto14753-chess-app.herokuapp.com

## How it works ?
You can set your nick, send direct and open request for 1-min, 5-min, 15 min games and check your match history.
